Zillow Estimates

A data mining activity of zillow estimates of housing prices. This is written in Python on Jupyter Notebook.

GitHub

First, I loaded the data set using read_feather. I also chose what columns I will based my model off of. The data types of each of the columns is listed.

Next, I cleaned the data and converted the columns to int or float types. Sold on is the only object as that column is used to split the data.

Next, I converted any area column that contained acres or sqft. I converted all measurements to sqft. Then I removed any abnormal data. As you can see, there were 9801 records remaining.

Lastly, I trained and tested my model. As you can see I got an RMSE of prediction of 0.0052785.
